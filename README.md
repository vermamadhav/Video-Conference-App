# video conference

A Flutter Project similar to zoom functionalities

## Features

- [x] Register/Login Screens
- [x] Create Meeting
- [x] Join Meeting
- [x] Function to mute all video and audio
- [x] Screen Sharing
- [x] And many functionalities
